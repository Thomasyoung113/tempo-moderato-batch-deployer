Quick Setup

Get deploying in under a minute

1

Clone Repository

git clone https://github.com/Thomasyoung113/tempo-moderato-batch-deployer.git
2

Install Dependencies

cd tempo-moderato-batch-deployer && npm install
3

Configure Environment

cp .env.example .env && echo "PRIVATE_KEY=your_key" > .env
4

Deploy!

npm run auto
